What is Angularjs

Angularjs is a framework that is primarily used to build single-page web applications. It works with long-established web technologies like HTML, CSS, and JavaScript to make the developing web apps easier and faster.

Why Angularjs?

Its not just a library, its a framework. Before Angularjs there were other JavaScript libraries which sort of did what Angularjs did, but Angularjs took all these awesome concepts like, data binding, DOM manipulation, Ajax services, Dependency Injection, Browser History, Testing and more, and put it into this framework.
